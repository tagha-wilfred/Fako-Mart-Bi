# 📊 Fako Mart BI System

A Google Apps Script–powered Business Intelligence (BI) reporting system that transforms raw supermarket data into automated, structured, and downloadable business reports.

This project simulates a real-world ERP-style analytics platform for a fictional supermarket (**Fako Mart**) using Google Sheets as the database and Apps Script as the backend engine.

---

# 🚀 Features

## 📄 Automated Report Generation
- Employee Performance Reports
- Manager Reports (expandable)
- Client & Invoice Reports
- Executive Owner Reports
- Operations Reports (extensible)

## 👤 Employee Intelligence System
- KPI-based performance scoring
- Attendance, efficiency, communication tracking
- Sales + errands contribution analysis
- Risk classification (Low / Medium / High)

## 🧠 Insight Engine
- Automatic performance summaries
- Strengths and weaknesses detection
- Operational recommendations
- Business intelligence insights

## 🖥️ Dynamic Sidebar Control Panel
- Employee selection (by name, not ID)
- Month & Year filtering
- Fully dynamic dropdowns from Google Sheets
- One-click report generation

## 📄 Professional Report System
- HTML-based document-grade reports
- Multi-section BI layouts
- Print-ready / PDF-compatible
- Real enterprise-style formatting
